using tabs in vim

Laurent Perrinet

2013-03-25 13:36

Source

  • call vim to open more files

    vim -p file1 file2 file3

  • or open one in your editing session

    :tabf <pattern>

    (will find the file corresponding to <pattern>)

  • to switch tab, use gt
